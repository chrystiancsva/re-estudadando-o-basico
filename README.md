# Fundamentos Front-End

Recentemente percebi que estava utilizando ferramentas de IA com tanta frequência que alguns conceitos básicos acabavam ficando menos presentes no meu dia a dia.

Por isso, criei este repositório para revisitar e praticar novamente os fundamentos do desenvolvimento web, começando do zero e reconstruindo cada conceito na prática.

## Tecnologias

- HTML
- CSS
- JavaScript
- TypeScript
- React
- Git
- Jest
- Cypress
- Playwright

## Objetivo

Não aprender algo novo, mas fortalecer aquilo que já faz parte da base de qualquer desenvolvedor.

A ideia é manter os fundamentos sempre afiados, entender o funcionamento das ferramentas em profundidade e evitar depender exclusivamente de soluções geradas por IA.

> Quem domina os fundamentos consegue aprender qualquer tecnologia com mais facilidade.
